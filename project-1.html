Healthy Vegan Exchange with Solidity, React and Redux.

Category : Blockchain
Client : Localhost
Completion : January 2021
Role : Solidity Programmer

During this project I learnt the basics of blockchain development.

I used the following technologies:

  • Ganache - for local ethereum network.
  • Solidity - for writing the smart contracts.
  • Truffle - to deploy smart contracts.
  • React - to have the core front-end application.
  • Bootstrap - to easily build the front end.
  • Redux - for updateing in real time the data from blockchain.

I gained knowledge in the following areas:

  • How to interact with Ethereum network.
  • How to keep the state in "a third party container", which updates the data.
  • Writing smart contracts.
  • Testing the smart contracts.
  • Ganache on Ubuntu, running on Windows.
  • React, Bootstrap and CSS.
